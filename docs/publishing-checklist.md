# Publishing Checklist

Before publishing the GitHub repository:

- [ ] Run every SQL query on Dune.
- [ ] Confirm all table names and column names still work.
- [ ] Create visualizations for each query.
- [ ] Add screenshots to `screenshots/`.
- [ ] Add live Dune dashboard link to `README.md`.
- [ ] Replace any placeholder values with actual Dune output.
- [ ] Add a short project description to the GitHub repo.
- [ ] Pin the repository on GitHub profile.
- [ ] Optional: write a LinkedIn post or X thread using the same thesis.
